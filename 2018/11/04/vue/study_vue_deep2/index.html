a) йқҷжҖҒжҲҗе‘ҳ

  1. йқҷжҖҒжҲҗе‘ҳеҢ…жӢ¬йқҷжҖҒеұһжҖ§е’ҢйқҷжҖҒж–№жі•гҖӮ
  2. ES2015еҸӘи§„е®ҡдәҶйқҷжҖҒж–№жі•пјҢжІЎжңүи§„е®ҡйқҷжҖҒеұһжҖ§гҖӮES2017жңүйқҷжҖҒеұһжҖ§зҡ„жҸҗжЎҲгҖӮзӣ®еүҚbabelе·Іж”ҜжҢҒиҪ¬з ҒгҖӮ
  3. йқҷжҖҒжҲҗе‘ҳжҳҜдҪңдёәзұ»зҡ„еұһжҖ§иҖҢеӯҳеңЁзҡ„пјҢе®ғеҸҜд»Ҙиў«еӯҗзұ»з»§жүҝпјҢдҪҶдёҚеҸҜд»Ҙиў«е®һдҫӢз»§жүҝеҸҠи®ҝй—®гҖӮ
  4. йқҷжҖҒжҲҗе‘ҳеҸӘиғҪйҖҡиҝҮзұ»еҗҚи®ҝй—®гҖӮ

еңЁdep.jsйҮҢпјҢзұ»Depе°ұжңүдёӘйқҷжҖҒеұһжҖ§targetгҖӮеңЁdep.jsеҸҠsrc/core/observer/index.jsйғҪеҸҜд»ҘзңӢеҲ°пјҢи®ҝй—®targetйғҪжҳҜйҖҡиҝҮDep.targetиҝҷдёӘзұ»еҗҚжң¬иә«еҺ»и®ҝй—®зҡ„пјҢжІЎжңүйҖҡиҝҮdep(dep = new Dep())еҺ»и®ҝй—®гҖӮ

еңЁdep.jsйҮҢEvan Youзҡ„жіЁйҮҠжҸҗеҲ°пјҢ targetиЎЁзӨәеҪ“еүҚжӯЈеңЁи®Ўз®—зҡ„зӣ®ж ҮwatcherгҖӮtargetжҳҜе…ЁеұҖе”ҜдёҖзҡ„пјҢеӣ дёәеңЁд»»дҪ•ж—¶й—ҙйғҪеҸӘжңүдёҖдёӘwatcherдјҡиў«и®Ўз®—гҖӮеә”иҜҘжҳҜеҮәдәҺжӯӨиҖғиҷ‘пјҢжүҖд»ҘEvan Youе°Ҷе…¶жіЁеҶҢдёәDepзҡ„йқҷжҖҒеұһжҖ§пјҢиҝҷж ·иҝҳеҸҜд»Ҙе…ЁеұҖи®ҝй—®пјҲзҢңжөӢпјҢдёҚиҙҹд»»дҪ•иҙЈд»»(в—ҸвҖҷв—ЎвҖҷв—Ҹ)пјүгҖӮ

targetеҜ№еә”зҡ„иҝҳжңүдёҖдёӘtargetStackгҖӮtargetStackеҸҠе°ҶдёҖзі»еҲ—йңҖиҰҒеӨ„зҗҶзҡ„targetеӮЁеӯҳе…Ҙж ҲгҖӮ

2гҖҒд»Јз Ғе®һзҺ°:

a) дҪ•ж—¶иҝӣиЎҢдҫқиө–ж”¶йӣҶпјҢдҪ•ж—¶йҖҡзҹҘдҫқиө–жӣҙж–°

DepдҪңдёәеҗ„Observerзҡ„и®ўйҳ…дёӯеҝғпјҢйҮҢйқўдҝқеӯҳзқҖи®ўйҳ…иҜҘObserverеҸҳеҢ–зҡ„watcherгҖӮ

еңЁдёӨдёӘең°ж–№дјҡз”ҹжҲҗж–°зҡ„depпјҢдёҖдёӘжҳҜеңЁObserverзҡ„constructorйҮҢпјҢдёҖдёӘжҳҜеңЁdefineReactiveйҮҢпјҲд»Јз ҒдҪҚдәҺcore/observer/index.jsпјүгҖӮ

defineReactiveз”ҹжҲҗзҡ„depжҳҜй”®еҗҚзҡ„depпјҢObserverз”ҹжҲҗзҡ„depжҳҜй”®еҖјзҡ„depгҖӮ

дҫӢеҰӮзӣ‘еҗ¬ obj = { a: 1 }

obj = {} ж—¶пјҢdefineReactiveз”ҹжҲҗзҡ„depеҺ»и§ҰеҸ‘дҫқиө–жӣҙж–°пјҢ obj.a = 2ж—¶з”ұObserverз”ҹжҲҗзҡ„depеҺ»и§ҰеҸ‘жӣҙж–°гҖӮ

ж•°з»„еҸҳеҢ–и§ҰеҸ‘дҫқиө–жӣҙж–°еҶҷеңЁvueж”№еҶҷзҡ„ж•°з»„ж–№жі•йҮҢ(core/observer/array.js)гҖӮйқһж•°з»„и§ҰеҸ‘дҫқиө–жӣҙж–°еңЁdefineReactiveзҡ„setйҮҢгҖӮ

жүҖжңүж•°жҚ®ж”¶йӣҶдҫқиө–зҡ„ең°ж–№ж”ҫеңЁдәҶdefineReactiveзҡ„getйҮҢгҖӮ

еҲҡејҖе§ӢжҲ‘иҝҳжңүзӮ№з–‘жғ‘пјҢзңӢиө·жқҘж•°з»„жІЎжңүиҝӣиЎҢдҫқиө–ж”¶йӣҶпјҢеӣ дёәеҪ“Observerзҡ„жҺҘж”¶зҡ„еҖјзұ»еһӢдёәж•°з»„ж—¶пјҢж №жң¬дёҚдјҡиҝӣе…ҘdefineReactiveгҖӮеҗҺйқўд»”з»ҶдёҖзңӢпјҢеҸ‘зҺ°еҗҺйқўиҝҳжңүдёҖеҸҘ


ж•°з»„зҡ„дҫқиө–ж”¶йӣҶе°ұжҳҜеңЁиҝҷйҮҢпјҢйҖҡиҝҮе…¶зҲ¶еҜ№иұЎдҪҝз”ЁжӯӨеҸҘиҝӣиЎҢж”¶йӣҶгҖӮеӣ дёәж•°з»„ж°ёиҝңжҳҜдҪңдёәжҹҗдёӘеҜ№иұЎзҡ„propertyиў«и®ҝй—®зҡ„пјҲжңҖејҖе§Ӣзҡ„ж•°жҚ®dataдёҖе®ҡжҳҜдёӘеҜ№иұЎпјҢдёҚеӯҳеңЁдёҖејҖе§Ӣи®ҝй—®зҡ„ж•°жҚ®зҡ„зұ»еһӢе°ұжҳҜж•°з»„иҝҷз§Қжғ…еҶөпјүгҖӮ
